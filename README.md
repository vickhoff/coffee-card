# Hyper Island coffee card

In the café at Hyper Island campus you can buy a coffee card with 10 slots that you check everytime you take a coffee. 

I lost my first coffe card the first day I bought it, so I decided to use my new knowledge in frontend development to create this minimal vanilla JavaScript web app built with Vite.

[Hosted at Vercel](https://coffee-card-plum.vercel.app/)

## Techniques used

- Vanilla Javascript 
- Animated variable fonts
- Local storage

## Getting Started

```bash
npm install
npm run dev
```
